### 如何部署

本项目已配置了Vercel部署文件，可以直接使用命令行部署：

```bash
# 安装Vercel CLI
npm install -g vercel

# 登录Vercel账户
vercel login

# 在项目根目录下部署
vercel --prod
```

部署完成后，CLI会显示项目的URL，通常格式为 `https://项目名称.vercel.app`
